# Core Features & Surfaces

## A. Gesture Library
- **Browse/Search**: Filter by tags, body area, intensity, duration.
- **Detail View**: Rich media (audio, video), instructions, contraindications.
- **Actions**: Favorite, Try (mini-player), Add to Flow.

## B. Flow Library
- **Curated Flows**: Bundled presets.
- **User Flows**: Local creation/storage.
- **Detail View**: Sequence, duration, intensity, equipment needed.

## C. Guided Flow Player (The "Engine")
- **Layout**: "Glanceable" Vertical Stack (Prev / **Current** / Next).
- **Current Card**: 
  - Massive countdown.
  - Minimal text.
  - Side indicator (L/R).
  - Media status.
- **Progress**: Segmented bar (one segment per step).
- **Controls**: Play/Pause (Primary), Back/Next, Repeat step.
- **Modes**: 
  - Audio-first (narration + cues).
  - Optional Video overlay.
  - "Glance Mode" (larger type).
- **State**: Persist resume state (snapshot).

## D. Flow Builder
- **CRUD**: Create/Edit/Save custom flows.
- **Editing**: Reorder steps, adjust duration/side/notes.
- **Preview**: Test in player.
