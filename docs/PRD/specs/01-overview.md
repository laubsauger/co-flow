# Product Overview

A mobile-first web app guiding massage sessions with reusable **gestures** chained into **flows**.
Optimized for "glanceability" as the user's hands will be busy.

## Core Value
- **Hands-Free Guidance**: Users can focus on the massage, not the screen.
- **Composable**: Gestures can be learned individually or chained into meaningful sessions.
- **Glanceable**: UI is designed to be read from a distance (phone on a table).
